# Intended Use

This repository is intended for public portfolio review by healthcare AI hiring managers, clinical AI product teams, clinical informatics teams, and AI evaluation teams.

The intended use is to show how synthetic or de-identified clinical material can be organized into evidence-bound human-AI review workflows.

Appropriate uses include:

- Reviewing portfolio artifacts
- Evaluating workflow design judgment
- Evaluating evidence-grounding and human review concepts
- Discussing non-clinical research prototypes in hiring conversations

This repository is not intended for clinical care, patient-facing use, diagnosis, treatment recommendation, prognosis, second opinions, medical device use, or deployment in a healthcare setting.
